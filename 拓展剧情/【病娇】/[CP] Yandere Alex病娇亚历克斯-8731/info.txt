Installation:
- Make sure you have "SMAPI", "Content Patcher", and "Unique Courtship" installed
- Unzip the file
- Open your "Stardew Valley\Mods" folder and drop the folder called "[CP] Yandere Alex Dialogue Expansion" there
- That's it!

NOTE: To see solely the yandere dialogue using "Unique Courtship", open the mod file and go to the config.json in "Unique Courtship" and change "enableAlex" from "enabled" to "disabled"
